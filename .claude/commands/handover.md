# セッション引き継ぎ（軽量版）

このセッションの作業内容を `.claude/handover.txt` のセッション履歴に簡潔に追記してください。

## 実行指示（重要）

**このコマンドは軽量・高速実行版です:**
- 思考プロセス（<thinking>タグ）を表示しない
- ファイル読み込みは `.claude/handover.txt` のみ
- プロジェクト静的情報は更新しない（generate_handover.sh実行なし）
- 詳細な分析は不要、結果だけを簡潔に記録
- 30秒以内で完了すること

## 実行手順

1. **既存の handover.txt を読み込む**
   - Readツールで `.claude/handover.txt` を読み込む

2. **「# 📚 セッション履歴」セクションを見つける**
   - このセクションが存在しない場合は、ファイル末尾に作成

3. **セッション履歴の末尾に簡潔な記録を追記**
   - Editツールで追記（または全体を書き直し）

## 追記する内容

以下の形式で簡潔に記録してください（3-5行程度）：

```markdown
## 💬 セッション [日時] (簡易)

- **完了**: [タスク1], [タスク2]
- **変更**: [ファイル名]
- **注意**: [重要な発見や決定事項]
- **次回**: [残タスクや次にやること]

---
```

## 記録内容

1. **完了したタスク**
   - 何を実装・修正したか（1行で）

2. **変更したファイル**
   - 主要なファイル名のみ

3. **注意点**
   - 重要な発見や決定事項（あれば）

4. **次回予定**
   - 残タスクや次にやること（あれば）

## 出力形式

- **サイズ**: 3〜5行程度の簡潔な記録
- **形式**: Markdown形式
- **追記先**: `.claude/handover.txt` の「# 📚 セッション履歴」セクション

## 使い分け

- **日常的な区切り**: `/handover` （このコマンド）
- **重要な作業後・完全記録**: `/handover-full`

詳細なプロジェクト解析とセッション内容の完全記録が必要な場合は `/handover-full` を使用してください。
